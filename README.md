# Azure AI-102 Certification Cheat Sheet

A comprehensive point-form reference for the **AI-102: Designing and Implementing a Microsoft Azure AI Solution** exam, organized for studying and memorization.

## 📄 Cheat Sheet

👉 [**View the full cheat sheet here**](https://github.com/kubakoz/azureAI102/blob/main/cheatshee.md)

## What's covered

- All 6 exam domains with their weightings
- Azure AI Services selection guide
- Security, authentication, and RBAC roles
- Responsible AI principles
- Generative AI & Azure OpenAI (GPT-4, embeddings, RAG)
- Agentic solutions & Foundry Agent Service
- Computer vision & custom models
- NLP: Language, Speech, Translator, CLU, Question Answering
- Azure AI Search pipeline (indexers, skillsets, knowledge stores, vector search)
- Azure AI Document Intelligence (prebuilt & custom models)
- Container deployment reference
- SDK patterns (Python & C#)
- Critical name changes and common exam pitfalls
